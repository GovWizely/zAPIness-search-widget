zAPIness Search Widget

Customizable and embeddable search widget that works with app.zapiness.com

Code and Docs on GitHub Star

Selected Fields show code

All fields will be shown if no fields are specified. 
          
            zAPI.SearchWidget.new({
              mountPoint: "#zAPI-sw-root",
              host: "https://rrsoft4.zapiness.xyz",
              endpoint: "sample_api",
              fields: [
                "agency",
                "description",
                "end_date",
                "supplier_name",
                "supplier_country",
                "supplier_city"
              ]
            });

Preview Mode show code

Show results once the search widget is loaded. 
          
            zAPI.SearchWidget.new({
              mountPoint: "#zAPI-sw-root",
              host: "https://rrsoft4.zapiness.xyz",
              endpoint: "sample_api",
              preview: true
            });

Specific Label show code

          
            zAPI.SearchWidget.new({
              mountPoint: "#zAPI-sw-root",
              host: "https://rrsoft4.zapiness.xyz",
              endpoint: "sample_api",
              label: "agency"
            });

Show All show code

          
            zAPI.SearchWidget.new({
              mountPoint: "#zAPI-sw-root",
              host: "https://rrsoft4.zapiness.xyz",
              endpoint: "sample_api",
              label: "agency",
              showAll: true
            });
